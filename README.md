
# Mobile measurement station





## Specification

- microcontoler STM32L476
- 2x DFRobot HR8833 
- Chassis Rectangle 4WD


## Pinout
 ### Microcontoler   
 #### Engine control
- PA0 PWM1/TIM2
- PA6 PWM2/TIM3
- PA1 LOGIC1
- PA5 LOGIC2
#### Remote Transmiter
- P send 1 (robot forward)
- P send 2 (robot left)
- P send 3 (robto right)
